# test

Runs all tests locally and in the test containers.

## Synopsis
```bash
$ joomla test [options]
```

## Arguments
This command has no arguments.

## Options
```
  -b, --basepath[=BASEPATH]  The path of the project root, defaults to current directory
```

## Description

Runs all tests locally and in the test containers
