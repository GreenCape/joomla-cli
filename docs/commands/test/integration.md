# test:integration

Runs integration tests on all test installations.

## Synopsis
```bash
$ joomla test:integration [options]
```

## Arguments
This command has no arguments.

## Options
```
  -b, --basepath[=BASEPATH]  The path of the project root, defaults to current directory
```

## Description

Runs integration tests on all test installations
