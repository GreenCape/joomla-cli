# quality:md

Generates pmd.xml using PHP MessDetector.

## Synopsis
```bash
$ joomla quality:mess-detect [options]
$ joomla quality:md
```

## Arguments
This command has no arguments.

## Options
```
  -s, --source[=SOURCE]  The source directory
  -l, --logs=LOGS        The logs directory [default: "build/logs"]
```

## Description

Generates pmd.xml using PHP MessDetector
