# docker:build

Generates the contents and prepares the test containers.

## Synopsis
```bash
$ joomla docker:build [options]
```

## Arguments
This command has no arguments.

## Options
```
  -b, --basepath[=BASEPATH]  The path of the project root, defaults to current directory
```

## Description

Generates the contents and prepares the test containers
