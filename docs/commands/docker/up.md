# docker:up

Starts the test containers after rebuilding them.

## Synopsis
```bash
$ joomla docker:up [options]
```

## Arguments
This command has no arguments.

## Options
```
  -b, --basepath[=BASEPATH]  The path of the project root, defaults to current directory
```

## Description

Starts the test containers after rebuilding them
