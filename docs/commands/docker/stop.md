# docker:stop

Stops and removes the test containers.

## Synopsis
```bash
$ joomla docker:stop [options]
```

## Arguments
This command has no arguments.

## Options
```
  -b, --basepath[=BASEPATH]  The path of the project root, defaults to current directory
```

## Description

Stops and removes the test containers
