# docker:start

Starts the test containers, building them only if not existing.

## Synopsis
```bash
$ joomla docker:start [options]
```

## Arguments
This command has no arguments.

## Options
```
  -b, --basepath[=BASEPATH]  The path of the project root, defaults to current directory
```

## Description

Starts the test containers, building them only if not existing
