# Install

Install a Joomla! extension.

**Synopsis:**
```bash
$ joomla install [options] [--] <extension>
```

**Arguments:**
```
  extension                The path to the extension.
```

**Options:**
```
  -b, --basepath=BASEPATH  The root of the Joomla! installation. Defaults to the current working directory. [default: "."]
 ```
